How to Use StockSage

1
Enter a Stock Symbol

Start by entering a valid stock symbol (like AAPL, MSFT, GOOGL) in the search box on the homepage.

2
Review Predictions

Once results are generated, you'll see predictions from three different algorithms, along with market sentiment analysis.

3
Analyze Charts

Examine the visual charts for each prediction algorithm to understand how they arrived at their forecasts.

4
View 7-Day Forecast

Check the 7-day price forecast table to see predicted values for the upcoming week.

5
Save to Portfolio

Add stocks to your portfolio to track them over time and quickly access predictions.

Technical Details

Data Source: Historical stock data is fetched from Yahoo Finance API with Alpha Vantage as a backup source.

Prediction Horizon: Next-day price predictions plus a 7-day forecast.

Market Sentiment: Analysis based on stock performance patterns.

Ensemble Method: Weighted average of multiple predictions, using inverse RMSE to give more weight to more accurate models.

Visualization: Dynamic charts generated using Matplotlib and styled for optimal readability.
